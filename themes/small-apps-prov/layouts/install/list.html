{{/* ── App Store ── */}}

App Store Recommended

The easiest way. Automatic updates, no computer needed. Provenance is 100% free on the App Store.

  • Works on iPhone, iPad, and Apple TV
  • Automatic updates via the App Store
  • Optional Provenance Plus for iCloud sync & beta access
{{/* ── AltStore ── */}}

AltStore

Sideload directly via AltStore. Tap "Add Source" to add the Provenance source, then install from the Browse tab.

  • Requires AltStore on your iPhone/iPad and AltServer on your Mac or PC
  • Refresh every 7 days (or use AltStore PAL on EU devices — no refresh needed)
  • Source URL: https://provenance-emu.com/apps.json

{{/* ── SideStore ── */}}

SideStore

Sideload wirelessly without a computer using SideStore. Tap "Add Source" to add Provenance.

  • Fully on-device — no Mac or PC required after initial setup
  • Refresh every 7 days using SideStore's built-in VPN trick
  • Source URL: https://provenance-emu.com/apps.json

IPA / Build from Source

Download the latest IPA and sideload manually, or build Provenance yourself from the open-source code.

  • Requires Xcode 16+ and an Apple Developer account (free or paid)
  • Full source available on GitHub under the Provenance License
  • See the build guide for step-by-step instructions
