## Bug Fixes

- **iPad skins fixed** - Skins now display and function correctly on iPad
- **Joystick in default skin fixed in landscape** - Landscape mode joystick input corrected for the default skin
- **Atari Lynx RetroArch controls fixed** - Controller mapping corrected for Atari Lynx emulation
- **RetroAchievements login fixed** - Login flow for RetroAchievements is now working properly
- **Skin selection and import UIs fixed** - Resolved issues with the skin browser and import interfaces
- **Skin rotation issues fixed** - Skins now handle device rotation smoothly
- **Skin previews fixed** - Preview images in the skin browser now load correctly
