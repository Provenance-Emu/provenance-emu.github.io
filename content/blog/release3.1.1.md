# Release 3.1.1 - Bug Fixes

Provenance 3.1.1 is a bug fix release focused on UI and settings improvements, fixing issues reported by the community after the 3.1.0 release.

## Links

- [App Store](https://apps.apple.com/us/app/provenance-app/id1596862805)
- [GitHub Release](https://github.com/Provenance-Emu/Provenance/releases/tag/3.1.1)
- [AltStore](/altstore/)

## Install Instructions

Download from the App Store for iOS or macOS Catalyst!

## What's Changed

### Settings Fixes
- Fixed settings titles display
- Fixed settings labels on small devices
- Replaced "Wifi" with "Wi-Fi" for correct terminology

### UI Improvements
- Fixed side menu icon alignment
- Fixed search bar position

### System Improvements
- Added check if app group container is readable for better stability
