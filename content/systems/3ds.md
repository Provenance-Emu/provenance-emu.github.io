Provenance brings the Nintendo 3DS to iPhone and iPad through the Azahar (formerly Citra) emulation core. The 3DS library spans nearly a decade of outstanding handheld gaming — Pokémon X/Y/Sun/Moon, The Legend of Zelda: A Link Between Worlds, Fire Emblem Fates, Monster Hunter 4 Ultimate, and hundreds more.

## Supported File Formats

- `.3ds` — Standard 3DS ROM format (most common)
- `.cia` — CTR Importable Archive (installed game format)
- `.cxi` / `.app` — 3DS executable formats
- `.ncsd` / `.ncch` — Raw NAND/card image formats

## Encryption Note

Many 3DS ROMs are encrypted. Provenance requires decrypted ROMs or proper AES keys to run encrypted titles. See the [wiki](https://wiki.provenance-emu.com/info/system-guides/3ds) for setup details.

## Features

- **Azahar Core** — Derived from Citra, offering broad 3DS game compatibility
- **Save States** — Snapshot support independent of in-game saves
- **Controller Support** — Full analog stick and button support via MFi, Xbox, and PlayStation controllers
- **Circle Pad Support** — Full analog input mapping including the Circle Pad Pro buttons
- **Touchscreen** — Bottom-screen touch input mapped to on-screen overlay

## iCloud Sync with Provenance Plus

[Provenance Plus](/plus/) syncs your 3DS save states and game library across all your Apple devices via iCloud.

## How to Get Started

1. **Download Provenance** from the [App Store](https://apps.apple.com/us/app/provenance-app/id1596862805) — it's free.
2. **Add your decrypted 3DS ROM files** via the Files app or Wi-Fi import.
3. **Start playing** — your 3DS games appear in your library automatically.

For detailed setup instructions and encryption key guidance, visit the [Provenance wiki](https://wiki.provenance-emu.com/info/system-guides/3ds).
