Provenance emulates the NEC PC Engine (sold as TurboGrafx-16 in North America) and its CD-ROM² add-on through the Mednafen core. Released in 1987 in Japan, the PC Engine punched well above its weight class with arcade-perfect ports and its revolutionary CD-ROM format. Highlights include Bonk's Adventure, Castlevania: Rondo of Blood, Ys Book I & II, and Street Fighter II'.

## Supported Systems

- **PC Engine / TurboGrafx-16** — HuCard cartridge games (`.pce`)
- **PC Engine CD-ROM² / TurboGrafx-CD** — CD-based games (`.ccd`, `.cue`, `.chd`)
- **SuperGrafx** — Enhanced PCE hardware (`.pce`, select titles)

## Supported File Formats

- `.pce` — PC Engine HuCard ROM
- `.cue` / `.ccd` / `.img` — CD image formats (use with CD-ROM games)
- `.chd` — Compressed Hunks of Data (recommended for CD images — smaller, faster)
- `.iso` — Raw ISO image (paired with audio tracks)

## BIOS Requirements

PC Engine CD-ROM² games require a BIOS file. See [BIOS Requirements](https://wiki.provenance-emu.com/installation-and-usage/bios-requirements) for the exact filename and placement instructions.

## Features

- **Mednafen Core** — Cycle-accurate PCE/PCE-CD emulation with excellent game compatibility
- **CHD Support** — Native CHD disc image support for CD-ROM games
- **Save States** — Full save state support
- **Controller Support** — Full MFi, Xbox, and PlayStation wireless controller support
- **6-Button Pad Support** — Extended controller mapping for Street Fighter II and other 6-button titles

## iCloud Sync with Provenance Plus

[Provenance Plus](/plus/) syncs your PC Engine save states and game library across all your Apple devices via iCloud.

## How to Get Started

1. **Download Provenance** from the [App Store](https://apps.apple.com/us/app/provenance-app/id1596862805) — it's free.
2. **Add your PC Engine ROM files** via the Files app or Wi-Fi import.
3. **For CD games**, add your BIOS file first — see [BIOS Requirements](https://wiki.provenance-emu.com/installation-and-usage/bios-requirements).
4. **Start playing** — your PC Engine games appear in your library automatically.

For detailed setup instructions, visit the [Provenance wiki](https://wiki.provenance-emu.com/).
