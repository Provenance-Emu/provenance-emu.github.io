The Sega Master System — released in Japan as the Mark III — was Sega's answer to the NES, featuring sharper graphics and a library that included Alex Kidd in Miracle World, Sonic the Hedgehog, Phantasy Star, and Golden Axe Warrior. While the NES dominated North America, the Master System found a devoted audience in Europe and Brazil. {{< brand >}}Provenance{{< /brand >}} brings the full Master System and Mark III library to your iPhone, iPad, and Apple TV.

{{< brand >}}Provenance{{< /brand >}} is free, open-source, and available directly from the App Store. No jailbreak required.

## Supported File Formats

Master System cartridges used straightforward ROM formats. {{< brand >}}Provenance{{< /brand >}} supports all common options:

- `.sms` — The standard Sega Master System ROM extension
- `.bin` — Generic binary format; many Master System and Mark III dumps use this extension
- `.rom` — Alternate ROM extension used by some archiving tools

Master System ROM files are small — typically under 512KB — making entire libraries easy to manage on a mobile device.

## Features

Playing Master System games in {{< brand >}}Provenance{{< /brand >}} gives you access to the full suite of emulator features:

- **Save States** — Save and resume anywhere; most Master System games used passwords rather than battery saves, making save states a major quality-of-life improvement
- **Controller Support** — Full MFi, Xbox, and PlayStation controller compatibility; the Master System's simple two-button layout maps cleanly to any modern controller
- **Shader Support** — CRT scanline and phosphor filters to recreate the look of the original hardware on a period-correct television
- **Cheat Codes** — Code support for supported titles
- **Fast Forward** — Speed through slower segments or repetitive sequences
- **On-Screen Controls** — Customizable touch overlay for playing without a physical controller
