Sega's Dreamcast was ahead of its time — online multiplayer, VMU memory cards, and an exceptional software library that included Shenmue, Sonic Adventure, Jet Set Radio, Marvel vs. Capcom 2, and Soul Calibur. Though the hardware was discontinued in 2001, the games hold up remarkably well. {{< brand >}}Provenance{{< /brand >}} brings the Dreamcast to your iPhone, iPad, and Apple TV.

{{< brand >}}Provenance{{< /brand >}} is free, open-source, and available directly from the App Store. No jailbreak required.

## Supported File Formats

Dreamcast games were GD-ROM disc-based. {{< brand >}}Provenance{{< /brand >}} supports the standard Dreamcast disc image formats:

- `.gdi` — GD-ROM Disc Image format; the most accurate representation of Dreamcast discs, typically paired with `.bin` track files in the same directory
- `.cdi` — DiscJuggler image format; widely used for Dreamcast backups
- `.chd` — Compressed Hunks of Data format; significantly reduces file sizes while maintaining full compatibility
- `.iso` — Standard disc image format for compatible releases

CHD is the recommended format for a Dreamcast library — Dreamcast games are large, and CHD compression makes them much more practical to store on a mobile device.

## Features

Playing Dreamcast games in {{< brand >}}Provenance{{< /brand >}} gives you access to the full suite of emulator features:

- **Save States** — Save anywhere, supplementing the Dreamcast's own VMU save system
- **VMU Emulation** — Virtual Memory Unit support for games that use the Dreamcast's built-in save functionality
- **Controller Support** — Full analog stick and trigger support via MFi, Xbox, and PlayStation controllers, closely matching the Dreamcast controller layout; on-screen touch controls also available
- **Shader Support** — CRT filters and upscaling options that make Dreamcast's distinctive art style shine on Retina displays
- **Cheat Codes** — Code support for Dreamcast titles
- **Fast Forward** — Skip loading screens or speed through slower sections
