The PlayStation 2 is the best-selling video game console ever made, and its library holds some of the greatest games ever created. {{< brand >}}Provenance{{< /brand >}} brings PS2 emulation to iPhone, iPad, and Apple TV — letting you play Grand Theft Auto: San Andreas, Shadow of the Colossus, God of War, Kingdom Hearts, Final Fantasy X, and hundreds of other titles on your Apple device.

## Supported File Formats

PlayStation 2 games were disc-based. {{< brand >}}Provenance{{< /brand >}} supports the most common PS2 disc image formats:

- `.iso` — Standard disc image format, the most widely used format for PS2 backups
- `.bin` / `.cue` — Common disc dump format; keep both files in the same directory
- `.chd` — Compressed Hunks of Data format; significantly reduces file sizes without sacrificing compatibility

CHD is recommended for large PS2 libraries — PS2 games can be several gigabytes each, and CHD compression makes storage much more manageable on a mobile device.

## Features

- **Save States** — Save anywhere, at any point in a game, not just at in-game memory card screens
- **Memory Card Emulation** — Virtual memory card support for games that use the PS2's standard save system
- **Controller Support** — Full analog stick and trigger support via MFi, Xbox, and PlayStation controllers; on-screen controls for touchscreen play
- **Shader Support** — Visual filters including upscaling options that make PS2-era graphics look sharper on modern Retina displays
- **Cheat Codes** — Code support for PS2 titles
- **Fast Forward** — Skip loading sequences or accelerate through slower sections

## Performance Note

PS2 emulation is computationally demanding. For the best experience, an iPhone 12 or newer (or equivalent iPad) is recommended. Older devices may experience reduced performance on graphically intensive titles.
