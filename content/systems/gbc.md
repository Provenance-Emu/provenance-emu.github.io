Provenance supports both the original Game Boy and the Game Boy Color on iPhone, iPad, and Apple TV. With hundreds of classic titles spanning two generations — Pokemon Red and Blue, The Legend of Zelda: Link's Awakening, Tetris, Pokemon Gold and Silver, Kirby's Dream Land, and countless others — the Game Boy library is one of the most beloved in gaming history.

Provenance is free on the App Store, open-source, and requires no jailbreak.

## Supported File Formats

- `.gb` — Original Game Boy ROM format
- `.gbc` — Game Boy Color ROM format

Game Boy Color is backward-compatible with original Game Boy cartridges, and Provenance handles both seamlessly. GBC-enhanced games that were playable on the original GB work correctly in Provenance as well.

## Features

- **Save States** — Save your progress at any moment, completely separate from in-game saves
- **Battery Save Emulation** — In-game save systems (like those in Pokemon) are fully emulated
- **Color Palettes** — For original Game Boy games, Provenance can apply color palettes similar to those used by the real Game Boy Color hardware
- **Controller Support** — Works with MFi, Xbox, and PlayStation controllers; on-screen touch controls available for handheld use
- **Fast Forward** — Speed through slower sections or dialogue
- **Cheat Codes** — Game Genie and GameShark code support

## iCloud Sync with Provenance Plus

[Provenance Plus](/plus/) adds iCloud sync for your Game Boy and Game Boy Color save states. Whether you're playing Pokemon on your iPhone or your iPad, your save data follows you automatically.

## How to Get Started

1. **Download Provenance** from the [App Store](https://apps.apple.com/us/app/provenance-app/id1596862805) — it's free.
2. **Add your Game Boy ROM files** (`.gb` for original, `.gbc` for Color) via iTunes File Sharing, the Files app, or web import.
3. **Start playing** — your Game Boy and Game Boy Color games appear in your library automatically.

For detailed setup instructions, visit the [Provenance wiki](https://wiki.provenance-emu.com/).
