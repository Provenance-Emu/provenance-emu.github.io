{{< brand >}}Provenance{{< /brand >}} brings the Nintendo GameCube to iPhone, iPad, and Apple TV through the Dolphin emulation core — the same battle-tested engine used on desktop platforms. The GameCube library is home to some of the greatest games ever made: The Legend of Zelda: The Wind Waker, Super Mario Sunshine, Metroid Prime, Resident Evil 4, and F-Zero GX.

## Supported File Formats

- `.iso` — Standard GameCube disc image
- `.gcm` — GameCube Master disc image (same format, different extension)
- `.gcz` — Dolphin-compressed GameCube image
- `.rvz` — Modern Dolphin compressed format (recommended — best compression)
- `.wbfs` — Wii Backup File System image (also accepted)
- `.ciso` — Compact ISO format

## Features

- **Dolphin Core** — Industry-standard GameCube/Wii emulation with broad game compatibility
- **Save States** — Snapshot and restore at any point, independent of memory card saves
- **Memory Card Emulation** — Virtual memory cards for in-game saves, no physical hardware needed
- **Controller Support** — Full MFi, Xbox, and PlayStation wireless controller support; analog sticks fully supported
- **Widescreen Support** — Many GameCube games support 16:9 widescreen mode
- **Folder Structure** — Organize your library using Dolphin's standard folder conventions

## Getting GameCube ROMs

GameCube games are large (typically 1–1.5 GB per disc). RVZ format offers the best compression while maintaining full compatibility. Use Dolphin on your Mac to rip your own GameCube discs.

See the [wiki guide on formatting ROMs](https://wiki.provenance-emu.com/installation-and-usage/roms/formatting-roms) for details on GameCube file organization.
