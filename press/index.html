About Provenance

Your Classic Games. Reimagined for iOS & tvOS.

Provenance is the most comprehensive open-source retro game emulator for Apple platforms. Supporting 38+ console systems from Atari to PlayStation 2, it delivers a museum-quality game library experience with native iOS and tvOS integration.

Full description

Provenance is a free, open-source multi-system emulator frontend for iOS and tvOS. Since its initial release in 2016, it has grown into the most comprehensive retro gaming solution available on Apple platforms, supporting over 38 console systems spanning six decades of gaming history.

Unlike other emulators, Provenance is built with a collector’s mindset. Every game in your library is presented with automatically downloaded box art, screenshots, release information, ratings, and even scanned manuals. The app feels native to Apple’s ecosystem, with support for Haptic feedback, Spotlight search, Siri shortcuts, iCloud sync, and full Siri Remote integration on Apple TV.

Provenance is the premier full-featured emulator for tvOS, bringing retro gaming to the living room with TopShelf integration, controller support for MFi, Xbox, and PlayStation gamepads, and 4K-ready upscaled graphics. Nearly all CD-based cores support CHD disc images; notably, Provenance’s native Mednafen core includes a custom CHD implementation not found in other standalone Mednafen forks, allowing PS1, Saturn, and PC-Engine CD libraries to use the compact CHD format without any loss of accuracy.

The project is 100% open source on GitHub with over 6,200 stars and an active community of contributors. It is free when sideloaded or built from source. An App Store version is also available with optional Provenance Plus features, while all core emulation functionality remains free.

Notable Features

Most Comprehensive System Support

38+ console systems from Atari 2600 to PlayStation 2, including rare systems like PC-FX, Virtual Boy, 3DO, and Neo Geo Pocket. Nearly all CD-based cores support CHD, including a custom CHD implementation in Provenance's native Mednafen core not found in other standalone Mednafen forks.

Premier Apple TV Emulator

The only emulator with full native tvOS support, TopShelf integration, Siri Remote optimization, and 4K-ready output.

Museum-Quality Game Library

Automatic box art, screenshots, manuals, ratings, and release info for every game. A digital collection, not just a game list.

100% Open Source

Fully transparent code on GitHub. No hidden tracking, no data collection. Community-driven since 2016.

Actively Developed — New Features Regularly

Online cheat code lookup from the libretro database, an in-app controller guide, configurable CRT shaders, expanded cheat support across 10+ cores, and more landing continuously.

Free Forever When Sideloaded

All features are free when sideloaded or built from source. No subscriptions, no paywalls, no tiers.
